grilla-admin
======
Admin tool for La Grilla's API

## Getting Started
1. Checkout the repo

2. Install dependencies with `npm install`

3. Start in debug mode with `npm start`

## Workflow

#### Development
```bash
npm start
```

#### Testing
```bash
npm test
```

#### Production
```bash
npm run build
```

## Dependencies
- [NodeJS](http://nodejs.org/)
